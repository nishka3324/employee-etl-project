# Healthcare Analytics ETL Project

## Overview
This project demonstrates an end-to-end ETL pipeline using Python, PostgreSQL, SQL, and Tableau.

## Technologies Used
- Python
- Pandas
- PostgreSQL
- SQLAlchemy
- Tableau Public
- GitHub

## Workflow
CSV File
→ Data Cleaning (Python/Pandas)
→ PostgreSQL Database
→ SQL Analysis
→ Tableau Dashboard
